# 项目 README

本页为项目概览摘要；仓库根目录 `README.md` 含完整说明。

## 快速开始

```bash
npm install
npm run dev
```

访问 http://localhost:5173

## 在线地址

- **站点**：部署根路径 `/`
- **文档**：`/docs/`（VitePress）

## 主要脚本

| 命令 | 说明 |
|------|------|
| `npm run dev` | 开发站点 |
| `npm run docs:dev` | 开发文档 |
| `npm run build:vercel` | Vercel 生产构建（站点 + 文档） |

详见 [DEPLOYMENT](./DEPLOYMENT.md)。

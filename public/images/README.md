## Directory layout

```
public/images/
├── marketing/   # brand & homepage imagery
├── pages/       # page-specific photos
└── products/    # catalog hero + lifestyle shots
```

To replace images: drop new JPEGs into the matching folder keeping filenames, or update paths in `app/lib/assets/paths.ts` and `app/data/productCatalog.ts`.
